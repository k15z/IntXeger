IntXeger¶

Generate unique strings from regular expressions.

Features¶

  • … coming soon …

Quick Start¶

import intxeger

xeger = intxeger.build("[a-z]{4}-[a-z]{4}-[a-z]{4}-[a-z]{4}")
xeger.sample(10) # generate 10 unique UUIDs
xeger.length()   # get the number of unique strings
xeger.get(1001)  # get the 1001th unique string

Installation¶

 Development Head: pip install git+https://github.com/k15z/intxeger.git
